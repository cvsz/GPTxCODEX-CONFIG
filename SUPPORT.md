# Support

## Supported channels

- Use GitHub Issues for reproducible defects and actionable feature requests.
- Use GitHub Discussions for setup questions and general usage help when Discussions is enabled.
- Use private vulnerability reporting for security issues.

## Before requesting help

1. Read `README.md` and the relevant configuration comments.
2. Confirm the exact Codex CLI or ChatGPT Desktop version.
3. Reproduce the issue using a clean configuration where practical.
4. Remove all credentials, tokens, private paths, hostnames, and proprietary data.
5. Search existing issues and pull requests.

## Information to include

- operating system and shell
- Codex or ChatGPT version
- relevant sanitized configuration
- exact commands
- expected and observed behavior
- sanitized logs and error messages

## Scope

Community support is provided on a best-effort basis. There is no guaranteed response time, service-level agreement, or warranty. Environment-specific MCP packages, cloud accounts, third-party services, and private infrastructure remain the user's responsibility.
