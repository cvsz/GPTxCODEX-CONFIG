# Kubernetes Prompt

Design, review, or troubleshoot Kubernetes workloads using production practices.

## Requirements
- Define namespaces, service accounts, RBAC, network policies, secrets, and pod security.
- Set requests, limits, probes, disruption budgets, topology spread, affinity, and autoscaling.
- Use immutable images, non-root execution, read-only filesystems, and explicit security contexts.
- Plan configuration, rollout strategy, rollback, migrations, jobs, and graceful termination.
- Establish metrics, logs, traces, events, alerts, SLOs, and operational runbooks.
- Validate manifests with schema checks, policy checks, dry runs, and staged deployment.

Avoid cluster-wide privileges and production mutations without explicit approval.
