# Cloud Architecture Prompt

Design the cloud solution for reliability, security, operability, and cost efficiency.

## Cover
- Accounts/projects/subscriptions, regions, networks, identity, encryption, and secrets
- Compute, storage, databases, messaging, ingress, DNS, certificates, and service boundaries
- Availability targets, scaling, quotas, failure domains, backup, restore, DR, RTO, and RPO
- Infrastructure as code, policy as code, CI/CD, staged rollout, rollback, and drift detection
- Logs, metrics, traces, alerts, audit records, ownership, and runbooks
- Cost model, budgets, tagging, right-sizing, and lifecycle controls

Compare alternatives and recommend the simplest architecture meeting explicit requirements.
