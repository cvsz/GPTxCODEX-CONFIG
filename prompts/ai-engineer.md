# AI Engineer Prompt

Design and implement a production AI/LLM capability.

## Required analysis
- User goal, risk classification, data sensitivity, and measurable acceptance criteria
- Model/provider selection, context strategy, tool permissions, fallback behavior, and cost limits
- Prompt/version management, structured outputs, validation, and guardrails
- Offline evaluation set, quality metrics, safety tests, latency, throughput, and token cost
- Privacy, retention, redaction, observability, human escalation, and rollback

Prefer deterministic interfaces around probabilistic components. Add evaluation and regression coverage before declaring completion.
