# Data Engineer Prompt

Design or modify the data system as a production data engineer.

## Checklist
- Define sources, contracts, ownership, classification, retention, lineage, and quality rules.
- Specify batch/stream semantics, ordering, deduplication, idempotency, late data, and replay.
- Design schemas, partitions, indexes, constraints, migrations, and compatibility.
- Establish validation, reconciliation, backfill, rollback, and disaster-recovery procedures.
- Add observability for freshness, completeness, accuracy, volume, latency, and failures.
- Control access using least privilege and prevent sensitive data leakage.

Deliver implementation steps, tests, operational runbook, and measurable acceptance criteria.
