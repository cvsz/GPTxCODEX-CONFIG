# ChatGPT + Codex Ultimate Professional Pack (2026)

This repository contains the packaged Windows Desktop / Native Windows configuration bundle for ChatGPT + Codex.

## Download

Use `ChatGPT-Codex-Ultimate-Professional-Pack-2026.zip` from the repository root.

## Contents

- Codex `config.toml`
- Role-specific profiles
- Reusable engineering prompts
- MCP integration templates
- VS Code configuration
- PowerShell lifecycle scripts
- Project templates

## Security

Review every MCP integration before enabling it. Do not commit API keys, access tokens, passwords, certificates, or connection strings.

## Status

Several optional MCP, PowerShell, and prompt files inside the ZIP are starter templates/placeholders and require environment-specific implementation and validation before production use.
